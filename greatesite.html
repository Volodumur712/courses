Створення по черзі як в уроках

Перші кроки по створенню сайта

  • Встанови та налаштуй XAMPP.
  • Відредагувати файли через блокнот
  • Створи в корені папки віртуального сервера файл index.php
  • Для того, щоб писати PHP код, необхідно створити файл із розширенням php (ім’я_файлу.php)
  • Додай до нього базову html-структуру і в тезі body створи блок з кодом php.
  • Усі змінні PHP необхідно починати з символу «$». Наприклад, $fruit.
  • echo щоб вивести дані
  • конкантинація крапочками
  • PHP функція var_dump() — ця функція повертає структуровану інформацію про те, що ти передав. Ця інформація включає тип і значення.
  • $_GET — масив параметрів, переданих скрипту через URL.
  • Метод GET означає, що потрібно отримати певний ресурс (файл, HTML, CSS, JS…).

  • Налаштування БД

    • Зайди в phpmyadmin та створи базу даних
    • Створи таблицю (users)
    • При створенні бази даних необхідно вказувати кодування utf8-general_ci.
    • Під час створення таблиці вказувати назви потрібно латиницею, у нижньому регістрі букв.
    • поле ID тип INT і галочку автопродовження (Чек бокс з підписом «A_I»)
    • поле назв і іншого тексту VARCHAR 255
  • Для підключення до MySQL із PHP нам треба вказати налаштування підключення: адресу сервера, логін, пароль, назву бази даних

Робота з формулами

  • створити формулу створення нового користувача
  • створити формулу видалення користувача
  • Перш ніж написати SQL запит у коді, перевір його правильність в PHPMyAdmin вкладка SQL, впиши туди свій запит і виконай, якщо помилки немає і ти отримуєш правильний результат роботи, то можеш сміливо використовувати цей запит.
  • Прописуючи SQL запит у коді будь уважний з лапками, більшість помилок пов’язана саме з ними!
  • Зміни структуру свого проекту, додавши в корінь твого проекту папки configs і partials.
  • Винеси код для підключення бази даних у файл config/db.php.
  • Винеси хедер та футер у файли header.php та footer.php відповідно.
  • Виконай підключення всіх файлів проекту в index.php
  • Підключи jQuery та свої власні скрипти до проекту.
  • Додай обробку форми до файлу add_user.php.
  • При надсиланні форми виконай збереження введених даних до бази.
  • Додай таблицю в проект і Виведи дані про користувачів з бази у таблицю. (зберегти в окремий файл)
  • Відредагувати таблицю як потрібно по колонках та кнопках
  • Створи файл edit_user.php, додай до нього форму та код для її обробки.
